# agentic-playground
Practical experiments, templates, and patterns for building AI agent systems.

## Contents (evolving)
- `experiments/`: isolated prototypes and quick validations
- `failure-modes/`: examples of agent breakdowns and how to detect/mitigate them
- `evals/`: evaluation harnesses and benchmark notes for agent behavior
- `templates/`: starter structures for new agent projects
- `patterns/`: repeatable approaches and notes

## How to use
- Browse the folders and pick a starting point.
- Copy a template into a new project and iterate.
- Read the notes to understand tradeoffs and gotchas.
